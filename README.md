# Foundation - Fullstack Workout Tracker

Foundation är en fullstack-applikation designad för att hjälpa användare att logga sina träningspass, hantera träningsprogram och följa sina framsteg genom statistik och personliga rekord.

**🔗 [Live Demo på Render](https://foundation-frontend-mn4f.onrender.com)**

---

## Tekniskt Stack

Projektet är byggt med en monorepo-struktur och använder följande teknologier:

- **Frontend:** React, Vite, TypeScript och SCSS.
- **Backend:** Node.js, Express, TypeScript.
- **Databas:** MongoDB Atlas.
- **Deployment:** Render (Web Service för Backend, Static Site för Frontend).

---

## Lokal Installation

1. Klona repot:
   ```bash
   git clone https://github.com/CamillaLundahl/Foundation.git
   
2. Installera dependencies för både klient och server:
   ```bash
   # /client
   npm install
   # /server
   npm install

3. Miljövariabler: skapa en .env-fil i server-mappen med MONGO_URI och JWT_SECRET.

4. Starta:
   ```bash
   npm run dev
